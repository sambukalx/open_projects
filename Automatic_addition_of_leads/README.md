## Описание

Это приложение на **PyQt5**, которое:
- Объединяет несколько **Excel-файлов** по общим заголовкам.
- Выполняет очистку, форматирование и заполнение пропущенных значений.
- Корректирует поля **e-mail** и **телефона**.
- Сохраняет результат в **XLSX** и **CSV**.

## Требования:
Python 3.8+,
pandas,
xlsxwriter,
PyQt5.
