search_file.py
==============

Этот модуль предназначен для поиска XML файлов в указанной директории. В частности, он ищет файл `index.xml` и выводит его путь.

Функции
-------


.. function:: find_xml_files(start_path)

    Функция для поиска файла `index.xml`, начиная с указанной директории. Использует библиотеку `os` для рекурсивного обхода директорий.

    **Описание**

    1. Выводит сообщение о начале поиска с использованием цветного текста (библиотека `colorama`).
    2. Рекурсивно обходит директории, начиная с `start_path`.
    3. Если находит `index.xml` в текущей директории, сохраняет полный путь к этому файлу и заменяет обратные слеши на прямые.
    4. Выводит найденный путь к файлу.
    5. Выводит сообщение о завершении поиска.
    6. Возвращает найденный путь.

    **Аргументы**

    - **start_path** *(str)*: Начальная директория, с которой начинается поиск.

    **Возвращаемое значение**

    Путь к найденному файлу `index.xml`.

    **Пример использования**:

    .. code-block:: python

        from search_file import find_xml_files

        start_path = '/path/to/start/directory'
        xml_file_path = find_xml_files(start_path)
        print(f'Найденный файл: {xml_file_path}')

    **Внешние зависимости**:

    - `colorama`: Используется для цветного вывода сообщений в консоли.

Атрибуты
--------

.. attribute:: found_paths

    Глобальная переменная, которая хранит путь к найденному файлу `index.xml`.

Пример использования:
---------------------

Вот пример использования функции `find_xml_files` из `search_file.py`:

.. code-block:: python

    from search_file import find_xml_files

    start_path = '/path/to/start/directory'
    xml_file_path = find_xml_files(start_path)
    print(f'Найденный файл: {xml_file_path}')
