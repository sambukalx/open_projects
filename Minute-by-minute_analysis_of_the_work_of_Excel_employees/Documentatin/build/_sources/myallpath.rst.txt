myallpath.py
============

Этот модуль предоставляет функции для выбора файлов и директорий, необходимых для обработки данных.

Функции
-------

.. function:: select_files()

    Функция для выбора необходимых файлов и директории. Используется графический интерфейс, построенный на основе библиотеки `tkinter`, для выбора файлов и директорий. Возвращает пути к выбранным файлам и директории.

    **Описание**

    1. Создание главного окна и его скрытие.
    2. Вызов диалоговых окон для выбора файлов и директории.
    3. Вывод выбранных путей на экран и их возврат в виде кортежа.

    **Этапы выполнения функции**:

    1. **Выбор текстового файла для сотрудников**:

        Диалоговое окно для выбора текстового файла с данными о сотрудниках.

        .. code-block:: python

            people = filedialog.askopenfilename(
                title="Выберите файл с сотрудниками",
                filetypes=[("Text Files", "*.txt")]
            )

    2. **Выбор ZIP архива**:

        Диалоговое окно для выбора ZIP архива.

        .. code-block:: python

            zip_path = filedialog.askopenfilename(
                title="Выберите ZIP файл",
                filetypes=[("ZIP Files", "*.zip")]
            )

    3. **Выбор файла с историей звонков**:

        Диалоговое окно для выбора файла с историей звонков.

        .. code-block:: python

            zvonki = filedialog.askopenfilename(
                title="Выберите файл с историей звонков",
                filetypes=[("Excel Files", "*.xlsx")]
            )

    4. **Выбор директории для вывода итогов**:

        Диалоговое окно для выбора директории, в которую будут сохранены результаты обработки.

        .. code-block:: python

            output_excel_path = filedialog.askdirectory(
                title="Выберите директорию для итогов"
            )

    **Возвращаемое значение**

    Кортеж, содержащий пути к выбранным файлам и директории:

    - **people** *(str)*: Путь к текстовому файлу с данными о сотрудниках.
    - **zip_path** *(str)*: Путь к ZIP архиву.
    - **zvonki** *(str)*: Путь к файлу с историей звонков.
    - **output_excel_path** *(str)*: Путь к директории для сохранения результатов.

    **Пример использования**:

    .. code-block:: python

        people, zip_path, zvonki, output_excel_path = select_files()
        print(f"Файл с сотрудниками: {people}")
        print(f"ZIP архив: {zip_path}")
        print(f"Файл с историей звонков: {zvonki}")
        print(f"Директория для итогов: {output_excel_path}")

Атрибуты
--------

Нет постоянных атрибутов; все данные собираются через функции выбора файлов и директорий.
