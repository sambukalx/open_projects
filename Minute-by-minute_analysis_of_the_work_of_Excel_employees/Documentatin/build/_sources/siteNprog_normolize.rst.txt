siteNprog_normolize.py
======================

Этот модуль предоставляет функции для извлечения и нормализации данных из отчетов, связанных с сайтами и программами.

Функции
-------


.. function:: extract_report_data_prog(data)

    Функция для извлечения данных из отчета по программам. Использует входные данные и выполняет нормализацию для дальнейшей обработки.

    **Описание**

    1. Получает данные отчета по программам в виде строки или другого формата.
    2. Выполняет анализ данных, извлекает нужную информацию и нормализует ее.
    3. Возвращает нормализованные данные в виде удобного для обработки формата.

    **Аргументы**

    - **data** *(разный тип данных)*: Исходные данные отчета по программам.

    **Возвращаемое значение**

    Нормализованные данные отчета по программам.

    **Пример использования**:

    .. code-block:: python

        from siteNprog_normolize import extract_report_data_prog

        raw_data_prog = "raw program data"
        normalized_data_prog = extract_report_data_prog(raw_data_prog)
        print(f'Нормализованные данные по программам: {normalized_data_prog}')



.. function:: extract_report_data_site(data)

    Функция для извлечения данных из отчета по сайтам. Использует входные данные и выполняет нормализацию для дальнейшей обработки.

    **Описание**

    1. Получает данные отчета по сайтам в виде строки или другого формата.
    2. Выполняет анализ данных, извлекает нужную информацию и нормализует ее.
    3. Возвращает нормализованные данные в виде удобного для обработки формата.

    **Аргументы**

    - **data** *(разный тип данных)*: Исходные данные отчета по сайтам.

    **Возвращаемое значение**

    Нормализованные данные отчета по сайтам.

    **Пример использования**:

    .. code-block:: python

        from siteNprog_normolize import extract_report_data_site

        raw_data_site = "raw site data"
        normalized_data_site = extract_report_data_site(raw_data_site)
        print(f'Нормализованные данные по сайтам: {normalized_data_site}')


Пример использования:
---------------------

Вот пример использования функций `extract_report_data_prog` и `extract_report_data_site`:

.. code-block:: python

    from siteNprog_normolize import extract_report_data_prog, extract_report_data_site

    # Исходные данные
    raw_data_prog = "raw program data"
    raw_data_site = "raw site data"

    # Нормализация данных
    normalized_data_prog = extract_report_data_prog(raw_data_prog)
    print(f'Нормализованные данные по программам: {normalized_data_prog}')

    normalized_data_site = extract_report_data_site(raw_data_site)
    print(f'Нормализованные данные по сайтам: {normalized_data_site}')
