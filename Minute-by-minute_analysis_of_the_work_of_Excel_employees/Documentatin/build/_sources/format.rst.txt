format.py
=========

Этот модуль предоставляет функцию для форматирования Excel файлов.

Функции
-------

.. function:: format_excel_file(input_filepath, output_filepath)

    Функция для форматирования Excel файла с заданным входным и выходным путями.

    **Аргументы**

    - **input_filepath** *(str)*: Путь к входному Excel файлу, который нужно форматировать.
    - **output_filepath** *(str)*: Путь к выходному Excel файлу, куда будет сохранен отформатированный файл.

    **Описание алгоритма**:

    1. Открывает входной Excel файл.
    2. Применяет необходимые форматирования к содержимому файла, например, стилизация ячеек, изменение форматов данных и прочее.
    3. Сохраняет отформатированный Excel файл по указанному выходному пути.

    **Пример кода**:

    .. code-block:: python

        input_filepath = '/path/to/input/file.xlsx'
        output_filepath = '/path/to/output/file.xlsx'
        format_excel_file(input_filepath, output_filepath)

    **Вывод на консоль**:
    В случае успешного выполнения функции отформатированный файл будет сохранен по указанному выходному пути. В случае ошибок будут выведены соответствующие сообщения об ошибках.
