myalldata.py
============

Этот модуль предоставляет данные и конфигурации, используемые в различных частях проекта.

Атрибуты
--------

.. attribute:: url_mapping

    Словарь, который содержит сопоставление URL для различных сервисов. Этот атрибут используется для перенаправления запросов к соответствующим сервисам.

    **Пример**:

    .. code-block:: python

        url_mapping = {
            "service1": "https://api.service1.com",
            "service2": "https://api.service2.com",
        }



.. attribute:: replacements

    Словарь с заменами значений. Этот атрибут используется для замены текстовых шаблонов в данных.

    **Пример**:

    .. code-block:: python

        replacements = {
            "old_value1": "new_value1",
            "old_value2": "new_value2",
        }



.. attribute:: bit_replacements

    Словарь для замены битовых значений. Этот атрибут используется для манипуляции и замены битов в бинарных данных.

    **Пример**:

    .. code-block:: python

        bit_replacements = {
            0b0001: 0b0010,
            0b0100: 0b1000,
        }



.. attribute:: employee_timezones

    Словарь, содержащий часовые пояса сотрудников. Этот атрибут используется для корректного отображения времени в зависимости от местоположения сотрудников.

    **Пример**:

    .. code-block:: python

        employee_timezones = {
            "employee1": "UTC+2",
            "employee2": "UTC-5",
        }



.. attribute:: company_timezone

    Строка, представляющая часовой пояс компании. Этот атрибут используется как стандартный часовой пояс для всех операций, если часовой пояс сотрудника не указан.

    **Пример**:

    .. code-block:: python

        company_timezone = "UTC"
