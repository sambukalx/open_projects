excel_create.py
===============

Этот модуль предоставляет функции для создания Excel файла с листами отделов и сотрудников.

Глобальные Импорты
------------------

- `colorama`: Модуль для раскрашивания текста в консоли.
- `pandas`: Модуль для работы с данными в формате DataFrame и Excel.
- `os`: Модуль для работы с файловой системой.

Функции
-------


.. function:: create_department_employee_excel(mas_sotrudniki, output_excel_path)

    Функция для создания Excel файла с листами отделов и сотрудников.

    **Аргументы**

    - **mas_sotrudniki** *(dict)*: Словарь с данными о сотрудниках и отделах (формат: {отдел: [список сотрудников]}).
    - **output_excel_path** *(str)*: Путь к папке, где будет создан Excel файл.

    **Описание алгоритма**:

    1. Проверяет, что входные данные являются словарем.
    2. Определяет полный путь к файлу.
    3. Создает новый Excel файл.
    4. Добавляет листы для отделов и сотрудников:
       - Для каждого отдела создает лист с именами сотрудников.
       - Для каждого сотрудника создает отдельный лист с базовой информацией (в настоящий момент пустой).
    5. Возвращает полный путь к созданному Excel файлу.

    **Пример кода**:

    .. code-block:: python

        mas_sotrudniki = {
            'Отдел_1': ['Сотрудник_1', 'Сотрудник_2'],
            'Отдел_2': ['Сотрудник_3', 'Сотрудник_4']
        }
        output_excel_path = '/path/to/directory'
        create_department_employee_excel(mas_sotrudniki, output_excel_path)

    **Вывод на консоль**:
    При успешном выполнении функции будет напечатано сообщение о создании Excel файла и его местоположении. В случае ошибки будут выведены соответствующие сообщения об ошибках.
