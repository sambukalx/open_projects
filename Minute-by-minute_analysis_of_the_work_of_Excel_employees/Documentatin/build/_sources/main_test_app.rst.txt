main_test_app.py
================

Этот модуль предоставляет графическое приложение для обработки различных данных и взаимодействия с пользователем через GUI.

Классы, методы и атрибуты
-------------------------


.. class:: MultiLogger

    Класс для логирования данных одновременно в консоль и виджет текста.


    - **converter** : *тип атрибута неизвестен*

      Конвертер для преобразования данных для логирования.

    - **text_widget** : *тип атрибута неизвестен*

      Виджет текста для вывода данных в GUI.

    - **terminal** : *тип атрибута неизвестен*

      Терминал для вывода данных в консоль.


    - **__init__**

      Инициализирует объект `MultiLogger`.

      .. method:: __init__()

    - **write**

      Записывает данные в консоль и виджет текста.

      .. method:: write(data)

        **Аргументы**

        - **data** *(str)*: Данные для записи.

    - **flush**

      Очищает буфер.

      .. method:: flush()



.. class:: ProcessingThread

    Класс для параллельной обработки данных в отдельном потоке.


    - **progress** : *тип атрибута неизвестен*

      Атрибут для отслеживания прогресса.

    - **status_signal** : *тип атрибута неизвестен*

      Сигнал статуса выполнения.

    - **error_signal** : *тип атрибута неизвестен*

      Сигнал для обработки ошибок.

    - **stop_signal** : *тип атрибута неизвестен*

      Сигнал для остановки выполнения.

    - **zip_path** : *тип атрибута неизвестен*

      Путь к ZIP файлу.

    - **zvonki** : *тип атрибута неизвестен*

      Данные о звонках.

    - **output_excel_path** : *тип атрибута неизвестен*

      Путь к выходному Excel файлу.

    - **people** : *тип атрибута неизвестен*

      Данные о сотрудниках.


    - **__init__**

      Инициализирует объект `ProcessingThread`.

      .. method:: __init__()

    - **run**

      Запускает выполнение параллельной задачи.

      .. method:: run()

    - **check_file**

      Проверяет корректность файла.

      .. method:: check_file(file_path)

        **Аргументы**

        - **file_path** *(str)*: Путь к файлу для проверки.

    - **check_folder**

      Проверяет корректность папки.

      .. method:: check_folder(folder_path)

        **Аргументы**

        - **folder_path** *(str)*: Путь к папке для проверки.

    - **check_stop**

      Проверяет сигнал остановки.

      .. method:: check_stop()

    - **kill_excel_processes**

      Завершает все процессы Excel.

      .. method:: kill_excel_processes()

    - **stop**

      Останавливает выполнение параллельной задачи.

      .. method:: stop()

Пример использования класса MainWindow:
---------------------------------------

Вот пример использования классов, предоставленных в `main_test_app.py`:

.. code-block:: python

    from main_test_app import MainWindow
    import sys
    from PyQt5.QtWidgets import QApplication

    if __name__ == "__main__":
        app = QApplication(sys.argv)
        main_window = MainWindow()
        main_window.show()
        sys.exit(app.exec_())
