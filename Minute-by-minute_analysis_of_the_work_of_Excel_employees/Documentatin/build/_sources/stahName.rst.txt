stahName.py
============

Этот модуль предоставляет функции для работы с папками и Excel-файлами. В частности, он включает функции для переименования папок на основании данных из Excel, проверки и модификации Excel-файла, удаления невалидных папок и обработки набора папок.

Функции
-------


.. function:: rename_folders_from_excel_cell(excel_path, folder_path)

    Функция для переименования папок на основании значений из ячеек Excel-файла.

    **Описание**

    1. Открывает Excel-файл по заданному пути.
    2. Читает значения из указанного столбца/листа.
    3. Переименовывает папки в указанной директории на основании прочитанных значений.
    4. Возвращает лог или статус выполнения операции.

    **Аргументы**

    - **excel_path** *(str)*: Путь к Excel-файлу.
    - **folder_path** *(str)*: Путь к папке, в которой находятся переименовываемые папки.

    **Возвращаемое значение**

    Лог или статус выполнения операции переименования.

    **Пример использования**:

    .. code-block:: python

        from stahName import rename_folders_from_excel_cell

        excel_file = '/path/to/data.xlsx'
        folder_directory = '/path/to/folders'
        rename_folders_from_excel_cell(excel_file, folder_directory)



.. function:: check_and_modify_excel(excel_path)

    Функция для проверки и модификации Excel-файла на соответствие определенным условиям.

    **Описание**

    1. Открывает и читает Excel-файл.
    2. Выполняет проверку данных на соответствие определенным условиям.
    3. При необходимости вносит изменения и сохраняет файл.
    4. Возвращает лог или статус выполнения операции.

    **Аргументы**

    - **excel_path** *(str)*: Путь к Excel-файлу, который необходимо проверить и модифицировать.

    **Возвращаемое значение**

    Лог или статус проверки и модификации.

    **Пример использования**:

    .. code-block:: python

        from stahName import check_and_modify_excel

        excel_file = '/path/to/data.xlsx'
        check_and_modify_excel(excel_file)



.. function:: remove_folder_if_invalid(folder_path)

    Функция для удаления папки, если она не соответствует определенным условиям.

    **Описание**

    1. Проверяет папку на соответствие определенным условиям.
    2. Если условия не выполнены, удаляет папку.
    3. Возвращает лог или статус выполнения операции.

    **Аргументы**

    - **folder_path** *(str)*: Путь к проверяемой папке.

    **Возвращаемое значение**

    Лог или статус выполнения операции удаления.

    **Пример использования**:

    .. code-block:: python

        from stahName import remove_folder_if_invalid

        folder_to_check = '/path/to/folder'
        remove_folder_if_invalid(folder_to_check)



.. function:: process_folders(base_path)

    Функция для обработки набора папок. Может включать переименование, проверку и удаление папок.

    **Описание**

    1. Обходит набор папок в указанной директории.
    2. Выполняет различные операции, такие как переименование, проверка и удаление.
    3. Возвращает лог или статус выполнения операций.

    **Аргументы**

    - **base_path** *(str)*: Путь к базе папок для обработки.

    **Возвращаемое значение**

    Лог или статус выполнения операции обработки папок.

    **Пример использования**:

    .. code-block:: python

        from stahName import process_folders

        base_directory = '/path/to/base/folder'
        process_folders(base_directory)


Пример использования:
---------------------

Вот пример использования функций `rename_folders_from_excel_cell`, `check_and_modify_excel`, `remove_folder_if_invalid` и `process_folders`:

.. code-block:: python

    from stahName import (rename_folders_from_excel_cell, check_and_modify_excel,
                          remove_folder_if_invalid, process_folders)

    # Переименование папок на основании данных из Excel
    excel_file = '/path/to/data.xlsx'
    folder_directory = '/path/to/folders'
    rename_folders_from_excel_cell(excel_file, folder_directory)

    # Проверка и модификация Excel-файла
    check_and_modify_excel(excel_file)

    # Удаление невалидных папок
    folder_to_check = '/path/to/folder'
    remove_folder_if_invalid(folder_to_check)

    # Обработка набора папок
    base_directory = '/path/to/base/folder'
    process_folders(base_directory)
