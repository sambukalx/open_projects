Analyze OP for minute
=====================

Проект **Analyze OP for minute** предназначен для автоматизации процессов обработки данных сотрудников, их активности, использования программ, посещений сайтов и звонков. Основная цель проекта — нормализация и экспорт данных в Excel для их последующего анализа.
