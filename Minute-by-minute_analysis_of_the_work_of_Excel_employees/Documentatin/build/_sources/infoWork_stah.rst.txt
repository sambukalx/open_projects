infoWork_stah.py
================

Этот модуль предоставляет функции и атрибуты для обработки информации о сотрудниках и их рабочем времени.

Функции и Атрибуты
------------------


.. attribute:: info_work_stah

    Глобальный атрибут, представляющий информацию о рабочем времени сотрудников.


.. function:: get_all_employees(directory)

    Функция для получения списка всех сотрудников из заданного каталога.

    **Аргументы**

    - **directory** *(str)*: Путь к директории, содержащей файлы с данными о сотрудниках.

    **Описание алгоритма**:

    1. Сканирует указанную директорию на наличие файлов с данными о сотрудниках.
    2. Собирает и возвращает список всех сотрудников, найденных в файлах.

    **Пример кода**:

    .. code-block:: python

        directory = '/path/to/employee/data'
        employees = get_all_employees(directory)
        print(employees)


.. function:: process_worktime_xlsx(file_path)

    Функция для обработки Excel файла с данными о рабочем времени сотрудников.

    **Аргументы**

    - **file_path** *(str)*: Путь к Excel файлу с данными о рабочем времени.

    **Описание алгоритма**:
    1. Загружает Excel файл с данными о рабочем времени сотрудников.
    2. Обрабатывает данные и обновляет информацию о рабочем времени в глобальном атрибуте :attr:`info_work_stah`.

    **Пример кода**:

    .. code-block:: python

        file_path = '/path/to/worktime/file.xlsx'
        process_worktime_xlsx(file_path)


.. function:: process_entrexit_xlsx(file_path)

    Функция для обработки Excel файла с данными о входах и выходах сотрудников.

    **Аргументы**

    - **file_path** *(str)*: Путь к Excel файлу с данными о входах и выходах.

    **Описание алгоритма**:

    1. Загружает Excel файл с данными о входах и выходах сотрудников.
    2. Обрабатывает данные и обновляет информацию о входах и выходах в глобальном атрибуте :attr:`info_work_stah`.

    **Пример кода**:

    .. code-block:: python

        file_path = '/path/to/entrexit/file.xlsx'
        process_entrexit_xlsx(file_path)


.. function:: process_acttime_xlsx(file_path)

    Функция для обработки Excel файла с данными об активном времени сотрудников.

    **Аргументы**

    - **file_path** *(str)*: Путь к Excel файлу с данными об активном времени.

    **Описание алгоритма**:

    1. Загружает Excel файл с данными об активном времени сотрудников.
    2. Обрабатывает данные и обновляет информацию об активном времени в глобальном атрибуте :attr:`info_work_stah`.

    **Пример кода**:

    .. code-block:: python

        file_path = '/path/to/acttime/file.xlsx'
        process_acttime_xlsx(file_path)


.. function:: scan_folders(base_directory)

    Функция для сканирования базовой директории с целью найти и обработать все связанные файлы с данными сотрудников.

    **Аргументы**

    - **base_directory** *(str)*: Путь к базовой директории для сканирования.

    **Описание алгоритма**:

    1. Сканирует базовую директорию и поддиректории на наличие файлов с данными сотрудников.
    2. Вызывает соответствующие функции для обработки каждого найденного файла (например, :func:`process_worktime_xlsx`).

    **Пример кода**:

    .. code-block:: python

        base_directory = '/path/to/base/directory'
        scan_folders(base_directory)


.. function:: print_pretty_info_work_stah()

    Функция для красивого вывода информации о рабочем времени сотрудников.

    **Описание алгоритма**:

    1. Форматирует информацию о рабочем времени из глобального атрибута :attr:`info_work_stah`.
    2. Печатает информацию в удобочитаемом формате.

    **Пример кода**:

    .. code-block:: python

        print_pretty_info_work_stah()


.. function:: print_pretty_info_work_stah1()

    Альтернативная функция для красивого вывода информации о рабочем времени сотрудников.

    **Описание алгоритма**:

    1. Форматирует информацию о рабочем времени из глобального атрибута :attr:`info_work_stah`.
    2. Печатает информацию в другом формате, отличающемся от :func:`print_pretty_info_work_stah`.

    **Пример кода**:

    .. code-block:: python

        print_pretty_info_work_stah1()

Переменные
----------

**info_work_stah** : *dict*
    Глобальный атрибут для хранения информации о рабочем времени сотрудников. Этот словарь обновляется функциями :func:`process_worktime_xlsx`, :func:`process_entrexit_xlsx`, и :func:`process_acttime_xlsx`.

**directory** : *str*
    Путь к директории, содержащей файлы с данными о сотрудниках. Эта переменная используется в функции :func:`get_all_employees` для сканирования файлов и извлечения информации о сотрудниках.

**file_path** : *str*
    Путь к Excel файлу, который необходимо обработать. Используется в функциях :func:`process_worktime_xlsx`, :func:`process_entrexit_xlsx`, и :func:`process_acttime_xlsx`.

**base_directory** : *str*
    Путь к базовой директории, которую необходимо сканировать для поиска файлов с данными сотрудников. Используется в функции :func:`scan_folders`.

Пример использования функций:
-----------------------------

Вот пример того, как можно использовать функции, предоставленные в `infoWork_stah.py`:

.. code-block:: python

    from infoWork_stah import get_all_employees, process_worktime_xlsx, scan_folders, print_pretty_info_work_stah

    # Пример использования функции get_all_employees
    directory = '/path/to/employee/data'
    employees = get_all_employees(directory)
    print(employees)

    # Пример использования функции process_worktime_xlsx
    file_path = '/path/to/worktime/file.xlsx'
    process_worktime_xlsx(file_path)

    # Пример использования функции scan_folders
    base_directory = '/path/to/base/directory'
    scan_folders(base_directory)

    # Пример использования функции print_pretty_info_work_stah
    print_pretty_info_work_stah()
