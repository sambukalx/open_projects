infoStah_toexcel.py
===================

Этот модуль предоставляет функции для форматирования строк времени и обновления Excel файлов с данными сотрудников.

Функции
-------


.. function:: format_time_string(time_string)

    Функция для форматирования строки времени в заданном формате.

    **Аргументы**

    - **time_string** *(str)*: Строка с временем, которую необходимо форматировать.

    **Описание алгоритма**:

    1. Получает строку времени на входе.
    2. Применяет форматирование к строке (например, изменение формата отображения времени).
    3. Возвращает отформатированную строку времени.

    **Пример кода**:

    .. code-block:: python

        time_string = '12:34:56'
        formatted_time = format_time_string(time_string)
        print(formatted_time)  # Вывод: '12:34'


.. function:: update_excel_with_employee_data(excel_path, employee_data)

    Функция для обновления Excel файла данными сотрудников.

    **Аргументы**

    - **excel_path** *(str)*: Путь к Excel файлу, который нужно обновить.
    - **employee_data** *(dict)*: Словарь с данными сотрудников, где ключи - это идентификаторы сотрудников, а значения - их данные.

    **Описание алгоритма**:

    1. Загружает указанный Excel файл.
    2. Обрабатывает данные сотрудников и обновляет соответствующие ячейки в Excel.
    3. Сохраняет обновленный Excel файл по тому же пути или в новый файл.

    **Пример кода**:

    .. code-block:: python

        excel_path = '/path/to/excel/file.xlsx'
        employee_data = {
            'emp1': {'name': 'John Doe', 'time': '12:34', 'department': 'IT'},
            'emp2': {'name': 'Jane Smith', 'time': '08:00', 'department': 'HR'}
        }
        update_excel_with_employee_data(excel_path, employee_data)
        print(f"Excel файл обновлен с данными сотрудников по пути: {excel_path}")

Переменные
----------

**excel_path** : *str*
    Путь к Excel файлу, который будет обновлен. Эта переменная указывает местоположение файла на диске.

**employee_data** : *dict*
    Словарь, содержащий данные о сотрудниках. Ключи словаря - это идентификаторы сотрудников (например, 'emp1', 'emp2'), а значения - это вложенные словари с информацией о каждом сотруднике. Например, данные могут включать имя сотрудника, время и отдел.

**time_string** : *str*
    Строка, представляющая время. Эта переменная используется функцией `format_time_string` для форматирования времени в определенном формате.

**formatted_time** : *str*
    Возвращаемое значение функции `format_time_string`. Эта переменная содержит отформатированную строку времени, готовую для использования.

**excel_data** : *dict*
    Словарь для хранения данных из Excel файла. Используется функцией `update_excel_with_employee_data` для обновления данных о сотрудниках в Excel файле.

Пример использования функций:
-----------------------------

Вот пример того, как можно использовать функции, предоставленные в `infoStah_toexcel.py`:

.. code-block:: python

    from infoStah_toexcel import format_time_string, update_excel_with_employee_data

    # Пример использования функции format_time_string
    time_string = '12:34:56'
    formatted_time = format_time_string(time_string)
    print(formatted_time)  # Вывод: '12:34'

    # Пример использования функции update_excel_with_employee_data
    excel_path = '/path/to/excel/file.xlsx'
    employee_data = {
        'emp1': {'name': 'John Doe', 'time': '12:34', 'department': 'IT'},
        'emp2': {'name': 'Jane Smith', 'time': '08:00', 'department': 'HR'}
    }
    update_excel_with_employee_data(excel_path, employee_data)
    print(f"Excel файл обновлен с данными сотрудников по пути: {excel_path}")
