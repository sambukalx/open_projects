sotrudniki.py
=============

Этот модуль предназначен для парсинга списка сотрудников и отделов из текстового файла и представления их в виде словаря.

Функции
-------


.. function:: parse_departments(people)

    Функция для чтения файла, содержащего отделы и их сотрудников, и возвращения словаря, где ключи - это названия отделов, а значения - списки сотрудников в этих отделах.

    **Описание**

    1. Открывает и читает файл с данными о сотрудниках и отделах.
    2. Парсит строки файла, определяя, какие строки относятся к отделам, а какие - к сотрудникам.
    3. Строит словарь, где ключи - названия отделов, а значения - списки сотрудников.
    4. Возвращает построенный словарь.

    **Аргументы**

    - **people** *(str)*: Путь к текстовому файлу, содержащему данные о сотрудниках и отделах.

    **Возвращаемое значение**

    Словарь, где ключи - это названия отделов, а значения - списки сотрудников в этих отделах.

    **Пример использования**:

    .. code-block:: python

        from sotrudniki import parse_departments

        file_path = '/path/to/people.txt'
        departments = parse_departments(file_path)
        print(f'Отделы и сотрудники: {departments}')


Переменные
----------


.. attribute:: departments

    Локальная переменная внутри функции `parse_departments`. Хранит промежуточные значения в процессе парсинга данных о сотрудниках и отделах.


.. attribute:: current_department

    Локальная переменная внутри функции `parse_departments`. Хранит название текущего отдела во время парсинга.

Пример использования:
---------------------

Вот пример использования функции `parse_departments`:

.. code-block:: python

    from sotrudniki import parse_departments

    file_path = '/path/to/people.txt'
    departments = parse_departments(file_path)
    print(f'Отделы и сотрудники: {departments}')


Зависимости
-----------


Этот модуль использует библиотеку `colorama` для цветного вывода сообщений в консоли.
