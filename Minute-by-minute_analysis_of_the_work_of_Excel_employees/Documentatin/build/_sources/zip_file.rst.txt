zip_file.py
===========

Этот модуль предназначен для распаковки zip-архивов в указанную директорию.

Функции
-------


.. function:: unzip_file(zip_path)

    Функция для распаковки содержимого zip-архива в директорию с именем архива.

    **Описание**

    1. Получает путь к zip-файлу и извлекает директорию файла и имя файла без расширения.
    2. Определяет путь для новой папки, в которую будет распакован архив.
    3. Создает папку с именем архива, если эта папка еще не существует.
    4. Разархивирует содержимое zip-файла в созданную папку.
    5. Заменяет обратные слеши на прямые в пути к папке для совместимости.
    6. Возвращает путь до распакованной папки.

    **Аргументы**

    - **zip_path** *(str)*: Путь к zip-файлу, который нужно распаковать.

    **Возвращаемое значение**

    Строка, содержащая путь до распакованной папки.

    **Пример использования**:

    .. code-block:: python

        from zip_file import unzip_file

        zip_file_path = '/path/to/archive.zip'
        extracted_folder = unzip_file(zip_file_path)
        print(f'Архив распакован в папку: {extracted_folder}')


Переменные
----------


.. attribute:: dir_name

    Локальная переменная внутри функции `unzip_file`. Хранит директорию, в которой находится zip-файл.


.. attribute:: folder_name

    Локальная переменная внутри функции `unzip_file`. Хранит имя zip-файла без расширения.


.. attribute:: output_folder

    Локальная переменная внутри функции `unzip_file`. Хранит путь к создаваемой папке, куда будет распакован архив.


.. attribute:: start_path

    Локальная переменная внутри функции `unzip_file`. Хранит путь до распакованной папки с замененными обратными слешами на прямые.


Пример использования:
---------------------

Вот пример использования функции `unzip_file`:

.. code-block:: python

    from zip_file import unzip_file

    zip_file_path = '/path/to/archive.zip'
    extracted_folder = unzip_file(zip_file_path)
    print(f'Архив распакован в папку: {extracted_folder}')


Зависимости
-----------

.. attribute:: colorama

    Этот модуль использует библиотеку `colorama` для цветного вывода сообщений в консоли.

.. attribute:: os

    Модуль `os` используется для работы с файловой системой, включая создание директорий и работу с путями файлов.

.. attribute:: zipfile

    Модуль `zipfile` используется для работы с zip-архивами, включая их распаковку.
